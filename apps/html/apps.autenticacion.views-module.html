iniciar_sesion(request)

source code 

Vista de inicio de sesion * Si el usuario inicia sesion con exito, retorna iniciar_sesion.html * Si el usuario esta inactivo, retorna no_activo.html * Si el usuario no exite en el sistema, retorna sesion_error.html

* Variables

  • formulario: es el formulario que el usuario debe completar para iniciar sesion
  • usuario_actor: es el usuario que realiza la accion
  • clave: es la clave secreta introducida por el usuario_actor
  • acceso: contiene el resultado de la funcion authenticate que lleva como parametro el par(usuario, contrasenha) verificando su existencia y estado en el sistema
Decorators:
  • @csrf_protect
